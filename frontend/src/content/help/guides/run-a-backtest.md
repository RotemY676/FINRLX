This guide walks through configuring and running a backtest from the **Backtests** page. It assumes you know which universe and which engine you want to test — see [Universe and features](/help/concepts/universe-and-features) and [Agents and engines](/help/concepts/agents-and-engines) if you are still deciding.

By the end you will have an experiment in the queue, results in the equity-curve panel, and a provenance row in the audit trail.
