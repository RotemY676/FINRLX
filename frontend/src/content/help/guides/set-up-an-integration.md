Integrations are how FINRLX pulls external data. The **Integrations** page groups them by category — Fundamentals, Market Data, News — and shows the connection status of each. Some integrations require API credentials; others are managed centrally and just need to be enabled.

This guide walks through enabling an integration, testing the connection, and reading the per-integration freshness chip.
