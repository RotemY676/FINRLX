A recommendation is FINRLX's central artifact. It bundles a set of portfolio weights, the evidence behind them, the governance state, and the risk overlays into one inspectable object. This walk-through opens a sample recommendation and points out what each panel means.

You will not need to make any changes to your live workspace — every step is read-only.
