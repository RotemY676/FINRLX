This tour walks through the main surfaces of FINRLX in about five minutes. By the end you will know what each top-bar element does, where decisions live, and how to drop into deeper screens when you want to learn more.

The tour assumes you have signed in and finished the welcome wizard. If you have not, start with [Understanding your profile](/help/getting-started/understanding-your-profile).
