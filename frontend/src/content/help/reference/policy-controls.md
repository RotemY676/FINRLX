A policy control is a named parameter that constrains recommendations. The most common controls are **CASH_FLOOR** (minimum portfolio % held in cash), **CONFIDENCE_FLOOR** (minimum data / model / operational confidence), exposure caps per name and per sector, and the turbulence threshold.

This reference page lists every policy control by name, with a one-paragraph description, the default value, the allowed range, and the place in the UI where you edit it.
