The Policies page is where you read and edit policy controls — the named parameters that constrain recommendations. The most consequential ones are CASH_FLOOR (minimum cash %) and CONFIDENCE_FLOOR (minimum data / model / operational confidence).

This reference catalogues the sections of the page and links to the full [policy controls](/help/reference/policy-controls) catalog.
