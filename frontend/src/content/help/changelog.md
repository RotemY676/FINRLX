The changelog records every user-facing change, newest first. Entries are tagged using the Keep-a-Changelog conventions: **Added** for new features, **Changed** for behavior changes, **Fixed** for bug fixes, **Deprecated** for things on the way out, and **Removed** for features that have been pulled.

The first populated changelog entry covers the launch of the Help center itself.
