# FINRLX

Private decision-intelligence platform for medium-term investing.

## Root structure
- DOCS: product, UX/UI, architecture, QA, and handoff documents
- frontend: UI application
- backend: API and business logic
- data: raw, processed, reference, and export data
- design: UX/UI work products
- scripts: setup, maintenance, and utility scripts
- infra: deployment and environment configuration
